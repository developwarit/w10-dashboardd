# Product

## Register

product

## Users

เจ้าหน้าที่และผู้เกี่ยวข้องกับงานซ่อมบำรุง W10 ที่ต้องดูภาพรวมสถานะงานจาก Google Sheet ในบริบทการทำงานจริง หน้าจอต้องอ่านเร็ว เปรียบเทียบตัวเลขง่าย และไม่ทำให้ผู้ใช้ต้องกลับไปไล่ข้อมูลในชีตเองเมื่อดู dashboard

## Product Purpose

EGAT Maintenance Dashboard แสดงข้อมูล W/O, สถานะงาน, งานเข้ารายสัปดาห์, งานเข้าตามกลุ่ม Equipment และรายการงานจากชีต Dashboard W10 All info จุดสำเร็จคือผู้ใช้เห็นภาพรวมได้ทันที เปลี่ยนตัวกรองได้ชัดเจน และตรวจรายการงานละเอียดได้ในแท็บรายงาน

## Brand Personality

ชัดเจน, น่าเชื่อถือ, ใช้งานจริง

## Anti-references

ไม่ควรเหมือนหน้า spreadsheet ที่แน่นจนอ่านยาก ไม่ใช้สีจัดหลายจุดจนรบกวนการเทียบตัวเลข ไม่ใช้ตัวหนังสือเล็กหรือสีจางเกินไป ไม่ให้ข้อความไทยเพี้ยนหรือตกขอบ และไม่ใส่ motion หรือ decoration ที่ไม่ช่วยงาน dashboard

## Design Principles

1. ข้อมูลสำคัญต้องเห็นก่อน: KPI, สถานะ และกราฟหลักต้องอยู่ในตำแหน่งสแกนง่าย
2. ตารางต้องอ่านได้จริง: ข้อความยาวต้องห่อหรือเลื่อนได้โดยไม่ทำให้ layout แตก
3. สีใช้เพื่อแยกความหมาย: ใช้สีช่วยอ่านสถานะและกลุ่มข้อมูล ไม่ใช้เป็น decoration หนัก
4. การกรองข้อมูลต้องชัดเจน: แยก action อ่านข้อมูลใหม่กับใช้ตัวกรองออกจากกัน
5. Dashboard ต้องนิ่งและเชื่อถือได้: refresh หน้าไม่ควรเปลี่ยน filter หรือข้อมูลในชีตเอง

## Accessibility & Inclusion

เน้น contrast อ่านง่าย, ฟอนต์ Sarabun สำหรับภาษาไทย, รองรับข้อความยาว, ไม่พึ่งสีอย่างเดียวในการตีความข้อมูล, หลีกเลี่ยง motion ที่ไม่จำเป็น และควรรองรับจอ desktop รวมถึง viewport แคบโดยไม่มีข้อความตกขอบ
