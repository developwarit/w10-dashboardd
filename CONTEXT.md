# Glossary

## OT Summary

The OT Summary is a daily overtime table for contract workers. It lists each worker by W group, records overtime hours for days 1-31, and summarizes holiday hours, 1.5x hours, pay amounts, total pay, and 3x pay.

## OT Error Check

The OT Error Check is a daily validation table for contract-worker overtime. Each worker has one status per day: pass when the sheet value is TRUE and error when the sheet value is FALSE. The row total is the number of error days.

## W Group

A W Group is the worker category used in the OT Summary. The current categories are W11, W12, W13, and W14.
